# Drift
A Chrome Extension 

Steps to Use:
1. Download and unzip the file above (click the 'Download ZIP' button under the green 'Code' button)
2. Open Chrome and go to chrome://extensions or click the puzzle piece icon in the top right tool bar
3. Toggle Developer mode on (top-right corner)
4. Click Load unpacked and select the drift folder
5. Open a new tab and you'll see Drift
